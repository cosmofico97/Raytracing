BRDFs and Pigments

Raytracing.UniformPigment — Type
UniformPigment <: Pigment(
    color::RGB{Float32} = RGB{Float32}(0.0, 0.0, 0.0)
)

A uniform pigment. This is the most boring pigment: a uniform hue over the whole surface.

See also: Pigment

source
Raytracing.CheckeredPigment — Type
CheckeredPigment <: Pigment
    color1::RGB{Float32} = RGB{Float32}(1.0, 1.0, 1.0),
    color2::RGB{Float32} = RGB{Float32}(0.0, 0.0, 0.0),
    num_steps::Int64 = 2
)

A checkered pigment. The number of rows/columns in the checkered pattern is tunable through the integer value num_steps, but you cannot have a different number of repetitions along the u/v directions.

See also: Pigment
